Questions

1. Why do we need a viewport meta tag?

tells browser how to control dimensions on different screens

2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

it will be the last code you put, so after you put "flex-direction: column;" then put "flex-direction: row;" then the "flex-direction: row;" will have priority

3. How do you change the CSS for a web page displayed on a phone?

Use media queries to apply specific styles when the screen width is below a certain size like phones or 300px for small phones and Test your design by resizing the browser window or using developer tools to inspect.
