Questions

1. Why is the viewport meta tag important?

this helps our website look good on different size screems and helps the browser know how to scale the page so that users can see and interact with it properly on smaller devices like phones and tablets.

2. How can you adjust the CSS for printing?

We can add another css file called print.css and put a link to it and then we use a rule called media print in this link. This lets you change styles when someone prints the page without affecting your original code.

3. When would you want to adjust the CSS for printing?

when you wanna print content on your website without the things that you dont need like the nav bar and other non useful things to print that will waste the users toner.
