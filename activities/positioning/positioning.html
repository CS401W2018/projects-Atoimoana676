Qutestions

1. What is normal flow in HTML?

Normal flow in HTML is just the way elements naturally appear on a page. Block elements, like <div>, stack on top of each other, while inline elements, like <span>, sit next to each other in a line.

2. What does float do? Why do you need to clear the float?

The float property lets you move an element to the left or right, so text or other content can wrap around it. But sometimes, floating elements mess up the layout, so we need to clear the float to make sure everything below it appears properly.

3. What do absolute, fixed, and relative do?

they help control where elements appear on a webpage. Relative moves an element slightly from its normal position. Absolute removes it from the normal flow and places it anywhere on the page and Fixed keeps the element in the same spot, even when scrolling.

4. How can you make nice navigation buttons for mobile devices?

we used css to apply the display: and flex; to keep them neat, border-radius or border bottom to make them look smooth, and padding to add space inside so the text moves inward.
