ASCII

it is a code for computers to understand human letters, numbers, and symbols.

Unicode

It is a code that allow computers to understand other human languages

UTF-8

UTF-8 is a way for computers to use Unicode to share all kinds of letters and symbols.

URL

A URL is the address of a website

Domain Name

A Domain Name is the name of a website thats easy to remember

HTTP

Its like a set of rules that computers use to talk to each other when you visit a website.

HTTPS

IP Address

An IP Address is a number that helps computers find each other on the Internet.

DNS

It takes the name of a website you type and finds the exact computer address where that website lives, so your computer can load it for you.
